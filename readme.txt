CG Scroll To Top is an awesome wordpress plugin to scroll top of your webpage with custom animations and lot's of customizations.

== Description ==

Are you looking for easy and best wordpress plugin to navigate top of your webpage with great animations ? 
CG Scroll To Top provides a way to customize your webpage scroll down pointing with colours and dial size. It provides a custom setting page in wordpress admin which is very simple. So, you can easily customize your scroller information part easily.
 
= More Information =


Visit the <a href="http://codersgod.com/web-plugins/cg-scroll-to-top">Coders God website</a> for documentation, support, and information.


== Installation ==



= From WordPress.org =

1. Download CG Scroll To Top.
2. Upload the 'cg-scroll-to-top' directory to your '/wp-content/plugins/' directory, using your favourite method (ftp, sftp, scp, etc...)
3. Activate CG Scroll To Top from your Plugins page.


= From WordPress Admin Panel =

1. Click on Add new under Plugin menu.
2. Type CG Scroll To Top in the input field and press enter.
3. Install and Activate CG Scroll To Top from listing.
